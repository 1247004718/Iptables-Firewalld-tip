# iptables的规则  
通和堵（包含转发）  

# iptables的规则链的大分类  
| PREROUTING | 路由前处理数据包 |  
|:----------:|:----------------:|  
| INPUT | 流入数据包 |  
| OUTPUT | 流出数据包 |  
| FORWARD | 处理转发数据包 |  
| POSTROUTIN | G路由后处理数据包 |  

# iptables的规则执行顺序  
从上到下  

# iptables一般会把来的流量的处理动作分成4类：  
| ACCEPT | 允许流量|  
|:------:|:-------:|  
| REJECT | 拒绝流量|  
| LOG | 记录日志|  
| DROP | 拒绝流量|  

# REJECT和DROP动作的区别  
| REJECT | 有返回|  
|:------:|:-------:|  
| DROP | 没有搭理|  

# iptables的bash命令生命周期
+ 重启失效  
+ 用命令可以永久生效  
    + `iptables-save`
